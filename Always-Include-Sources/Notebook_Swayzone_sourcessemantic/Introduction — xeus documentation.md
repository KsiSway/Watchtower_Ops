C++ implementation of the Jupyter Kernel protocol

Introduction

https://xeus.readthedocs.io/en/latest/#introduction

is a framework meant to facilitate the implementation of kernels for Project Jupyter. It takes the burden of implementing the Jupyter Kernel protocol so developers can focus on implementing the interpreter part of the kernel.
