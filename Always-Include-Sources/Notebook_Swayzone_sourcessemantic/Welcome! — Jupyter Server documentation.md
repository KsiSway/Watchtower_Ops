Introduction

https://jupyter-server.readthedocs.io/#introduction

Jupyter Server is the backend that provides the core services, APIs, and

REST endpoints

https://petstore.swagger.io/?url=https://raw.githubusercontent.com/jupyter-server/jupyter\_server/main/jupyter\_server/services/api/api.yaml

for Jupyter web applications.

Jupyter Server is a replacement for the Tornado Web Server in

Jupyter Notebook

https://github.com/jupyter/notebook

. Jupyter web applications should move to using Jupyter Server. For help, see the

Migrating from Notebook Server

https://jupyter-server.readthedocs.io/operators/migrate-from-nbserver.html#migrate-from-notebook

Applications

https://jupyter-server.readthedocs.io/#applications

Jupyter Server extensions can use the framework and services provided by Jupyter Server to create applications and services.

#### Examples of Jupyter Server extensions include:

Jupyter Lab

https://jupyterlab.readthedocs.io/

JupyterLab computational environment.

Jupyter Resource Usage

https://github.com/jupyter-server/jupyter-resource-usage

Jupyter Notebook Extension for monitoring your own resource usage.

Jupyter Scheduler

https://jupyter-scheduler.readthedocs.io/

Run Jupyter notebooks as jobs.

jupyter-collaboration

https://jupyterlab-realtime-collaboration.readthedocs.io/

A Jupyter Server Extension Providing Support for Y Documents.

https://nbclassic.readthedocs.io/

Jupyter notebook as a Jupyter Server extension.

Cylc UI Server

https://cylc.org/

A Jupyter Server extension that serves the cylc-ui web application for monitoring and controlling Cylc workflows.

For more information on extensions, see

Server Extensions

https://jupyter-server.readthedocs.io/developers/extensions.html#extensions

Who's this for?

https://jupyter-server.readthedocs.io/#who-s-this-for

The Jupyter Server is a highly technical piece of the Jupyter Stack, so we've separated documentation to help specific personas:

https://jupyter-server.readthedocs.io/users/index.html#users

: people using Jupyter web applications.

https://jupyter-server.readthedocs.io/operators/index.html#operators

: people deploying or serving Jupyter web applications to others.

https://jupyter-server.readthedocs.io/developers/index.html#developers

: people writing Jupyter Server extensions and web applications.

Contributors

https://jupyter-server.readthedocs.io/contributors/index.html#contributors

: people contributing directly to the Jupyter Server library.

If you finds gaps in our documentation, please open an issue (or better, a pull request) on the Jupyter Server
