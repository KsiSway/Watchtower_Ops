Kernels (Programming Languages)

Kernels (Programming Languages)

https://docs.jupyter.org/en/latest/projects/kernels.html#kernels-programming-languages

The Jupyter team maintains the

https://github.com/ipython/ipython

project which is shipped as a default kernel (as

https://docs.jupyter.org/en/latest/projects/kernels.html#term-ipykernel

) in a number of Jupyter clients. Many other languages, in addition to Python, may be used in the notebook.

The community maintains many other language kernels, and new kernels become available often. Please see the

list of available kernels

https://github.com/jupyter/jupyter/wiki/Jupyter-kernels

for additional languages and

kernel installation instructions

https://ipython.readthedocs.io/en/latest/install/kernel\_install.html

to begin using these language kernels.

https://docs.jupyter.org/en/latest/projects/kernels.html#kernels

Kernels are programming language specific processes that run independently and interact with the Jupyter Applications and their user interfaces.

https://github.com/ipython/ipykernel

is the reference Jupyter kernel built on top of

https://ipython.org/

, providing a powerful environment for interactive computing in Python.

jupyter-client

https://jupyter-client.readthedocs.io/en/stable/

contains the authoritative description of the Jupyter messaging protocol which is used by clients to communicate with the kernels.

https://xeus.readthedocs.io/en/latest/

facilitates the implementation of kernels for Jupyter and provides a number of kernels such as

https://github.com/jupyter-xeus/xeus-cling

https://github.com/jupyter-xeus/xeus-sql

(SQL) and many more.

https://ipython.org/

https://docs.jupyter.org/en/latest/projects/kernels.html#term-IPython

interactive computing in Python.
