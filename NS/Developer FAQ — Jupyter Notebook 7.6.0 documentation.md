How do I install a prerelease version such as a beta or release candidate?

You can install a prerelease version of the notebook using the

python -m pip install notebook --pre --upgrade

If you are using

, you can install a prerelease version of the notebook using the alpha or beta label. For example, to install the latest alpha release, you can run:

conda install -c conda-forge -c conda-forge/label/notebook\_alpha notebook=7.0.0a18
