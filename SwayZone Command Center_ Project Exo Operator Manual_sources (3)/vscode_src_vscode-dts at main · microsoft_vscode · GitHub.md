file in this directory, its name must follow this pattern

vscode.proposed.\[a-zA-Z\]+.d.ts

src/vs/platform/extensions/common/extensionsApiProposals.ts

(make sure to run

npm run watch

declare and implement your proposal

make sure to use the

checkProposedApiEnabled

isProposedApiEnabled

-utils to enforce the API being proposed. Make sure to invoke them with your proposal's name which got generated into

extensionsApiProposals.ts
