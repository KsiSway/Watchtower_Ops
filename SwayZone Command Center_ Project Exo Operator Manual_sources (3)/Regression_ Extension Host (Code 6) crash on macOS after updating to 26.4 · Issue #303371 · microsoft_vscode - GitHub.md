Performance Issue

After updating macOS to version 26.4, the VS Code Insiders Extension Host crashes repeatedly on startup with exit code 6. The logs indicate a failure in the ExtensionHostProcess.

The crash is resolved by setting "http.systemCertificatesNode": false in settings.json. This suggests a regression or incompatibility in how the current Insiders build (Electron/Node.js) interacts with the macOS system certificate store following the OS update.
